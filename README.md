## Description

[Safer] Back-end service repository.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## API Route available

```bash
# Authentication
$ /api/auth/register
$ /api/auth/login

# User (jwt token obliged)
$ /api/user/:id
```
